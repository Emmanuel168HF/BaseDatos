### 2. Descripción del Sistema de Base de Datos

El sistema administra:

- Empleados
- Departamentos
- Ubicaciones
- Proyectos
- Dependientes
- Asignación de proyectos

Permite administrar la información del personal, los departamentos, las ubicaciones, los proyectos y la participación de los empleados en la empresa.

## 4. Diccionario de Datos

******Tabla:** Employee

**Descripción:** _Almacena la información de los empleados._

| Campo | Tipo | Longitud | Restricciones | Descripción |
|:---|:---|:---:|:---|:---|
| SSN | CHAR | 11 | PK, NN | Identificador único del empleado. |
| FirstName | VARCHAR | 50 | NN | Nombre del empleado. |
| LastName | VARCHAR | 50 | NN | Apellidos del empleado. |
| Address | VARCHAR | 150 | NN | Dirección del empleado. |
| BirthDate | DATE | - | NN | Fecha de nacimiento. |
| Salary | DECIMAL | - | NN | Salario del empleado. |
| Sex | CHAR | 1 | NN | Sexo del empleado. |
| DeptNumber | INT | - | FK, NN | Departamento al que pertenece. |

### 5. Relaciones

| Columna 1 | Columna 2 | Columna 3 |
|:----------|:--------:|--------------------------------------:|
| Department -> | Employee | 1:N \| Un departamento tiene varios empleados. |
| Department -> | Locations | 1:N \| Un departamento puede tener varias ubicaciones. |
| Department -> | Projects | 1:N \| Un departamento administra varios proyectos. |
| Employee -> | Dependent | 1:N \| Un empleado puede tener varios dependientes. |
| Employee -> | Works_On | 1:N \| Un empleado puede trabajar en varios proyectos. |
| Projects -> | Works_On | 1:N \| Un proyecto puede tener varios empleados asignados. |

### 7. Integridad Referencial

| Regla | Descripción |
|:----------|:--------------------------------------|
| IR-01 | No se puede registrar un empleado en un departamento inexistente. |
| IR-02 | No se puede registrar un proyecto para un departamento inexistente. |
| IR-03 | No se puede registrar un dependiente para un empleado inexistente. |
| IR-04 | No se puede asignar un empleado a un proyecto inexistente. |
| IR-05 | No se puede eliminar un departamento mientras tenga empleados, proyectos o ubicaciones asociadas. |

---

### 8. Reglas del Negocio

- Todo empleado pertenece a un departamento.
- Cada departamento tiene un gerente asignado.
- Un departamento puede tener varias ubicaciones.
- Un departamento puede administrar varios proyectos.
- Un empleado puede participar en varios proyectos.
- Un proyecto puede tener varios empleados.
- Un empleado puede registrar varios dependientes.
- Las horas registradas en un proyecto deben ser mayores que cero.
